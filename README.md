# weather-forecast-system

💡Project Idea
A Weather Forecasting System is a software application that predicts future weather conditions using data like temperature, humidity, and wind speed. It processes this data to provide forecasts and alerts, helping users plan activities and make informed decisions.

⚙️ Features
*Add city weather data
*Predict weather using scoring model
*Display all cities
*Severe weather alerts (priority-based)
*Simulate hourly weather updates
*Save & load from file

🧠 Concepts Used
*Structures
*Conditional statements
*Functions
*Arrays
*Queue
